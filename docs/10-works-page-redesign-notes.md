# Works Page Redesign Notes

## Direction
- 活動と実績ページを、単なるリンク集ではなく Editorial Portfolio 型の実績ハブへ寄せる。
- 参考方向:
  - journalist portfolio: 代表実績を先に見せる
  - press/media page: 数字・掲載導線・問い合わせ導線を整理する
  - category hub: 媒体カテゴリ別に後から拡張できる構造にする

## Current Structure
1. Hero
   - 活動の立ち位置を明確化
   - 実績数と活動領域をサマリー表示
2. Selected Works
   - 代表的な導線を3件だけ強調
3. Expertise
   - 活動カテゴリごとの強みを説明
4. Archive
   - 既存リンク集を媒体カテゴリ別に整理
5. Next
   - Issue #15 の根拠付き実績追加に接続
6. Media Kit
   - Route: `#/media-kit`
   - 紹介・取材・依頼前に見られる簡易資料ページ

## Future Improvements
- Issue #15 の確認済みデータを `Specialized Works` として追加する。
- Featured items は採用根拠が固まり次第、媒体・記事単位で再選定する。
- Media Kit ページは Issue #15 の確認済みデータが増えたら、媒体掲載履歴と専門領域別実績を追加する。
