# Project Overview

## 1. Basic Info
- Project Name: Sensoria - 五感で紡ぐ、美の旅路 -
- Owner: Yuichi-TanakaJP
- Last Updated: 2026-03-08
- Target Release Date: Released (incremental updates)
- Current Status: `released`

## 2. Purpose
- Why this site exists: 個人の活動・経歴を集約し、外部メディアへの導線を一本化するため
- Business or personal goal: 「公開して育てる」運用で、ポートフォリオ資産を継続的に強化する
- What users should do after visiting: 記事を読む、プロフィール理解、外部リンク遷移、問い合わせ導線へ進む

## 3. Target Audience
- Primary audience: アート・旅・美容領域に関心のある読者、編集者、協業候補
- Secondary audience: 既存読者、SNS経由の新規訪問者
- Key user needs: 世界観の理解、実績確認（300+ articles）、過去記事へのアクセス

## 4. KPI
- Main KPI: 外部リンククリック数（記事導線、SNS導線）
- Sub KPI: スクロール到達率、主要CTAクリック率
- Measurement period: 週次確認（月次で傾向確認）
- Baseline / target: 初期はベースライン計測、3か月で主要CTA CTRの改善を目標化

## 5. Scope
- Included in this phase:
  - 1ページ構成のブランド表現
  - 基本ナビゲーション
  - Cloudflare Web Analyticsによる軽量計測
- Excluded in this phase:
  - CMS導入
  - 動的記事配信
  - 複雑なマーケティングオートメーション

## 6. Stakeholders
- Decision maker: Owner
- Implementer: Owner (with AI-assisted coding)
- Reviewer: Owner

## 7. Risks / Notes
- Major risks:
  - 画像がプレースホルダー（`picsum`）中心で、ブランド資産としては暫定
  - `mailto` とSNSリンクが暫定値で、本運用連絡先の反映が必要
- Open questions:
  - 問い合わせ導線をフォームにするか外部SNS導線で完結させるか
  - 記事一覧を静的維持するかCMS/MDX化するか
